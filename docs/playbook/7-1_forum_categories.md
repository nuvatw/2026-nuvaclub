# Forum: Post Types

> **Section ID:** `forum-categories`
> **Parent:** Forum
> **Purpose:** Understanding post categories

---

## TL;DR

5 post categories. Pick the right one when posting. Helps others find your content.

---

## Categories Overview

| Type | Color | When to Use |
|------|-------|-------------|
| **Discussion** | Blue | General chat, opinions |
| **Question** | Green | Need help with something |
| **Share** | Purple | Show off something cool |
| **Resource** | Orange | Helpful links, tools |
| **Announcement** | Red | Official news (admin only) |

---

## Discussion

### Purpose
General conversations, opinions, debates

### Examples
- "What AI tools do you use daily?"
- "Hot take: GPT-5 predictions"
- "Your favorite prompt engineering tips?"

### When to Use
- Sharing opinions
- Starting conversations
- Discussing trends
- Asking for opinions

---

## Question

### Purpose
Get help from the community

### Examples
- "How do I connect GPT to Zapier?"
- "Why is my prompt getting bad results?"
- "Best practices for RAG systems?"

### When to Use
- You're stuck on something
- Need specific help
- Looking for how-to guidance
- Troubleshooting issues

### Tips for Questions
- Be specific about the problem
- Include what you've tried
- Provide context
- Format code properly

---

## Share

### Purpose
Show off accomplishments, creations, wins

### Examples
- "I automated my entire workflow!"
- "Built this AI tool over the weekend"
- "Finally passed Level 10!"

### When to Use
- Completed a project
- Achieved something
- Made something cool
- Want feedback on work

---

## Resource

### Purpose
Share helpful links, tools, references

### Examples
- "Best free AI courses list"
- "Prompt library I found useful"
- "Comprehensive API documentation"

### When to Use
- Found something useful
- Compiled a list
- Want to share tools
- Providing educational content

### Tips for Resources
- Briefly describe what it is
- Explain why it's useful
- Credit original creators

---

## Announcement

### Purpose
Official platform news and updates

### Who Can Post
- Admin and moderators only

### Examples
- New feature releases
- Platform maintenance
- Policy updates
- Community events

---

## Quick Decision Guide

```
"I need help with something" → Question
"I want to chat about something" → Discussion
"I made/found something cool" → Share
"I have a useful link to share" → Resource
```
