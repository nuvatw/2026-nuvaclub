# Dashboard: My Courses

> **Section ID:** `dashboard-courses`
> **Parent:** Member Dashboard
> **Access:** Explorer and above

---

## TL;DR

Track your learning progress. See enrolled courses, continue where you left off, celebrate completions.

---

## My Courses View

### Enrolled Courses
All courses you've started watching, showing:
- Course thumbnail
- Course title
- Progress bar (% complete)
- Last watched date
- "Continue" button

### Completed Courses
Courses you've finished:
- Completion badge
- Date completed
- Option to rewatch

---

## Course Cards

Each course card shows:

```
┌─────────────────────────────┐
│ [Thumbnail]                 │
│ Course Title                │
│ ████████░░░░ 67%           │
│ Last watched: 2 days ago    │
│ [Continue Learning →]       │
└─────────────────────────────┘
```

---

## Actions

| Action | How |
|--------|-----|
| Continue course | Click "Continue" on any card |
| View course details | Click the course title |
| Remove from list | Click "..." → "Remove" |
| Mark as favorite | Click the star icon |

---

## Filtering & Sorting

### Filter Options
- All Courses
- In Progress
- Completed
- Not Started

### Sort Options
- Recently Watched
- Progress (high to low)
- Alphabetical
- Date Added

---

## Progress Tracking

### How Progress Works
- Progress = lessons watched / total lessons
- Each lesson marks complete when 90% watched
- Progress syncs across devices

### Resume Feature
Click "Continue Learning" to:
1. Open the course player
2. Jump to your last position
3. Continue from exact timestamp

---

## Tips

1. **Check daily** - See what's next
2. **Complete one course** before starting another
3. **Use favorites** to queue courses for later
4. **Celebrate completions** - You earned them!
