# Dashboard: Favorites

> **Section ID:** `dashboard-favorites`
> **Parent:** Member Dashboard
> **Access:** Explorer and above

---

## TL;DR

Quick access to saved content. Favorite courses, bookmark forum posts, save Sprint projects for later.

---

## Types of Favorites

| Type | Where to Favorite | Icon |
|------|-------------------|------|
| Courses | Learn page | Star |
| Forum Posts | Forum page | Bookmark |
| Sprint Projects | Sprint page | Star |

---

## Favorite Courses

Courses you've starred for later watching:
- See course thumbnail and title
- View your current progress (if any)
- One-click to start or continue
- Remove with star toggle

### How to Favorite a Course
1. Go to Learn page
2. Hover over any course card
3. Click the star icon
4. Course appears in Favorites

---

## Bookmarked Posts

Forum discussions you want to revisit:
- Post title and preview
- Category and author
- Date bookmarked
- Link to full post

### How to Bookmark a Post
1. Open any forum post
2. Click the bookmark icon
3. Post appears in Favorites

---

## Saved Projects

Sprint projects you found inspiring:
- Project thumbnail
- Title and creator
- Sprint it belongs to
- Link to full project

### How to Save a Project
1. Open any Sprint project
2. Click the star icon
3. Project appears in Favorites

---

## Managing Favorites

### Remove an Item
1. Find the item in Favorites
2. Click the star/bookmark icon again
3. Item is removed

### Organize
- Filter by type (Courses, Posts, Projects)
- Sort by date added
- Search within favorites

---

## Tips

1. **Favorite first, decide later** - Don't lose interesting content
2. **Review weekly** - Clean up what you no longer need
3. **Use for planning** - Queue content for upcoming learning sessions
4. **Don't over-favorite** - Keep it useful, not overwhelming
