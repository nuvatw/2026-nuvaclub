# Space: Get Matched

> **Section ID:** `space-matching`
> **Parent:** Space
> **Purpose:** How to find and connect with mentors/mentees

---

## TL;DR

1. Get Duo Ticket → 2. Go to Space → 3. Create/browse posts → 4. Send request → 5. Start learning

---

## Step by Step

### Step 1: Get a Duo Ticket
1. Go to Shop
2. Navigate to Duo Tickets
3. Buy Go, Run, or Fly
4. Space unlocks immediately

### Step 2: Go to Space
1. Click "Space" in navigation
2. You'll see three sections:
   - My Nunu
   - My Vava
   - Matching Board

### Step 3: Create or Browse

**Create a Post:**
1. Click "Create Matching Post"
2. Describe what you're looking for
3. Set your availability
4. Publish

**Browse Posts:**
1. Go to Matching Board
2. Use filters to narrow down
3. Read profiles carefully

### Step 4: Send Request
1. Find someone interesting
2. Click their profile
3. Review their information
4. Click "Request Match"
5. Add a message

### Step 5: Start Learning
1. Wait for acceptance
2. Once matched, you're connected
3. Start your mentorship journey
4. Use My Space dashboard to manage

## Creating a Great Post

### What to Include
- Your current skill level
- What you want to learn
- Your goals
- Availability (days/times)
- Preferred format (chat, video, etc.)

### Example Post
```
Looking for: Nunu with prompt engineering expertise

About me: I'm a marketer learning to use AI for content
creation. I've completed Lv.1-3 courses and want to
level up my skills.

Goals:
- Master advanced prompting
- Build automation workflows
- Create AI-powered content system

Availability: Weekday evenings (6-9pm EST)
```

## Finding the Right Match

### For Vavas (Finding a Nunu)

Look for:
- Relevant expertise
- Compatible availability
- Good reviews/ratings
- Clear communication style
- Matching goals

Red flags:
- Vague profiles
- No availability listed
- Poor feedback history

### For Nunus (Finding a Vava)

Look for:
- Clear goals
- Realistic expectations
- Active engagement
- Compatible schedule

Red flags:
- Unclear what they want
- Unrealistic demands
- No commitment shown

## After Matching

### First Week
- Introduce yourselves
- Set expectations
- Define goals
- Establish schedule

### Ongoing
- Regular check-ins
- Progress reviews
- Adjust as needed
- Provide feedback

### If It's Not Working
- Communicate openly
- Try to resolve issues
- End match gracefully if needed
- No penalty for ending
