# Forum: Formatting Tips

> **Section ID:** `forum-markdown`
> **Parent:** Forum
> **Purpose:** Markdown formatting for posts and comments

---

## TL;DR

Use Markdown to make your posts look good. Bold, italic, lists, code blocks, links, quotes - all supported.

---

## Text Formatting

### Bold
```markdown
**bold text**
```
Result: **bold text**

### Italic
```markdown
*italic text*
```
Result: *italic text*

### Bold + Italic
```markdown
***bold and italic***
```
Result: ***bold and italic***

### Strikethrough
```markdown
~~crossed out~~
```
Result: ~~crossed out~~

---

## Headers

```markdown
# Heading 1
## Heading 2
### Heading 3
```

Use headers to organize long posts.

---

## Lists

### Bullet List
```markdown
- Item one
- Item two
- Item three
```

Result:
- Item one
- Item two
- Item three

### Numbered List
```markdown
1. First step
2. Second step
3. Third step
```

Result:
1. First step
2. Second step
3. Third step

### Nested Lists
```markdown
- Main item
  - Sub item
  - Sub item
- Main item
```

---

## Code

### Inline Code
```markdown
Use `inline code` for short snippets
```

Result: Use `inline code` for short snippets

### Code Blocks
````markdown
```python
def hello():
    print("Hello, World!")
```
````

Result:
```python
def hello():
    print("Hello, World!")
```

### Syntax Highlighting
Specify the language after the opening backticks:
- `python`
- `javascript`
- `typescript`
- `json`
- `bash`

---

## Links

### Basic Link
```markdown
[link text](https://example.com)
```

Result: [link text](https://example.com)

### With Title
```markdown
[link text](https://example.com "Hover title")
```

---

## Quotes

### Block Quote
```markdown
> This is a quoted text
> It can span multiple lines
```

Result:
> This is a quoted text
> It can span multiple lines

### Nested Quote
```markdown
> Main quote
>> Nested quote
```

## Tables

```markdown
| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Cell 1   | Cell 2   | Cell 3   |
| Cell 4   | Cell 5   | Cell 6   |
```

Result:

| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Cell 1   | Cell 2   | Cell 3   |
| Cell 4   | Cell 5   | Cell 6   |

## Horizontal Rule

```markdown
---
```

Creates a divider line between sections.

## Quick Reference

| Format | Syntax | Result |
|--------|--------|--------|
| Bold | `**text**` | **text** |
| Italic | `*text*` | *text* |
| Code | `` `code` `` | `code` |
| Link | `[text](url)` | [text](url) |
| Quote | `> text` | > text |

---

## Tips

1. **Preview before posting** - Check how it looks
2. **Use code blocks** - For any code or prompts
3. **Break up long posts** - Use headers and sections
4. **Keep it readable** - Don't over-format
