# Test: Question Types

> **Section ID:** `test-types`
> **Parent:** Test
> **Purpose:** Understanding different question formats

---

## TL;DR

5 question types across the levels. Basic levels use simple formats. Expert levels use essays. Know what to expect.

---

## Question Types Overview

| Type | Levels Used | Example |
|------|-------------|---------|
| True/False | 1-3 | "GPT stands for Generative Pre-trained Transformer" |
| Multiple Choice | 1-6 | "Which is NOT a type of machine learning?" |
| Multiple Select | 3-8 | "Which ARE valid AI applications? (Select all)" |
| Short Answer | 4-9 | "What does GPT stand for?" |
| Essay/Practical | 7-12 | "Design a chatbot workflow for..." |

---

## True/False

### Format
> **Question:** Is ChatGPT a large language model?
> - True
> - False

### Tips
- Read carefully - one word can change meaning
- Don't overthink - usually straightforward
- Watch for absolute words ("always", "never")

### Used In
Levels 1-3 (Basic tier)

---

## Multiple Choice

### Format
> **Question:** Which is NOT a type of machine learning?
> - A) Supervised
> - B) Unsupervised
> - C) Reinforced
> - D) Caffeinated

### Tips
- Read all options before answering
- Eliminate obviously wrong answers
- Look for the "most correct" answer
- Watch for "NOT" or "EXCEPT" questions

### Used In
Levels 1-6 (Basic and Intermediate tiers)

---

## Multiple Select

### Format
> **Question:** Which ARE valid AI applications? (Select all that apply)
> - [ ] Image generation
> - [ ] Text summarization
> - [ ] Time travel
> - [ ] Language translation

### Tips
- There may be multiple correct answers
- Each option is evaluated independently
- Don't select unless you're sure
- Read ALL options carefully

### Used In
Levels 3-8 (Intermediate and Advanced tiers)

---

## Short Answer

### Format
> **Question:** What does GPT stand for?
>
> [Your answer here]

### Tips
- Be concise but complete
- Spelling matters (somewhat)
- AI checks for semantic similarity
- Include key terms

### Used In
Levels 4-9 (Intermediate and Advanced tiers)

---

## Essay/Practical

### Format
> **Question:** Design a customer service chatbot workflow. Include:
> - Input handling
> - Response logic
> - Fallback behavior
> - Success metrics
>
> [Long-form answer here]

### Tips
- Structure your answer clearly
- Use bullet points or sections
- Address all parts of the question
- Show your reasoning process
- Quality over quantity

### Used In
Levels 7-12 (Advanced and Expert tiers)
