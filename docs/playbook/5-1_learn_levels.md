# Learn: The 10 Levels

> **Section ID:** `learn-levels`
> **Parent:** Learn
> **Purpose:** Course difficulty progression system

---

## TL;DR

Courses are organized into **10 difficulty levels**. Lv.1-2 are beginner-friendly. Lv.9-10 are for experts. Start where you are, progress at your pace.

---

## Level Breakdown

| Level | Name | Description | Badge Color |
|-------|------|-------------|-------------|
| Lv.1 | Beginner | Introduction to AI concepts | Green |
| Lv.2 | Basic | Foundational skills | Green |
| Lv.3 | Elementary | Building understanding | Blue |
| Lv.4 | Intermediate | Applied knowledge | Blue |
| Lv.5 | Upper-Intermediate | Advanced applications | Default |
| Lv.6 | Advanced | Complex implementations | Default |
| Lv.7 | Proficient | Expert techniques | Orange |
| Lv.8 | Expert | Specialized skills | Orange |
| Lv.9 | Master | Industry-level mastery | Red |
| Lv.10 | Grandmaster | Elite expertise | Red |

---

## Level Badges

Courses display color-coded level badges:

| Color | Levels | Meaning |
|-------|--------|---------|
| **Green** | Lv.1-2 | Beginner-friendly, start here |
| **Blue** | Lv.3-4 | Elementary, building blocks |
| **Default** | Lv.5-6 | Intermediate, core skills |
| **Orange** | Lv.7-8 | Advanced, specialized |
| **Red** | Lv.9-10 | Master level, expert |

---

## What You Learn at Each Level

### Lv.1-2: Getting Started
- What is AI?
- Basic terminology
- Simple prompts
- Using ChatGPT/Claude

### Lv.3-4: Building Skills
- Prompt engineering basics
- Understanding model parameters
- Better outputs
- Common use cases

### Lv.5-6: Intermediate Applications
- Advanced prompting techniques
- Workflow automation
- API basics
- Tool integration

### Lv.7-8: Advanced Topics
- Multi-step workflows
- Custom agents
- Fine-tuning concepts
- Enterprise applications

### Lv.9-10: Expert Level
- AI strategy
- Security and governance
- Leadership applications
- Cutting-edge techniques

---

## Recommended Progression

```
Beginner Path:
Lv.1 → Lv.2 → Lv.3 → ...

Skip if experienced:
If you already know basics → Start at Lv.3-4
If you're a developer → Start at Lv.5-6
```

---

## Matching Test Levels

Each course level has a matching test level. Complete the course, take the test to prove your knowledge.

| Course Level | Test Level | Pass to Unlock |
|--------------|------------|----------------|
| Lv.1 | Test Lv.1 | Badge + Lv.2 recommended |
| Lv.2 | Test Lv.2 | Badge + Lv.3 recommended |
| ... | ... | ... |

---

## Tips

1. **Don't skip levels** - Foundations matter
2. **Review before advancing** - Make sure you understand
3. **Take the test** - Validate your learning
4. **Revisit if needed** - No shame in reviewing
