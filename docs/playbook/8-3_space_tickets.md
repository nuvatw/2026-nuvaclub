# Space: Which Ticket?

> **Section ID:** `space-tickets`
> **Parent:** Space
> **Purpose:** Choosing the right Duo Ticket tier

---

## TL;DR

- **Go** = First-time mentorship seekers (budget-friendly)
- **Run** = Serious learners wanting professional guidance
- **Fly** = VIPs wanting the absolute best (founder access)

---

## Quick Comparison

| Feature | Go | Run | Fly |
|---------|:--:|:---:|:---:|
| Space Access | Yes | Yes | Yes |
| Regular Nunus | Yes | Yes | Yes |
| Discord Community | Yes | Yes | Yes |
| Certified Nunus | No | Yes | Yes |
| Priority Queue | No | Yes | Yes |
| Founder Access | No | No | Yes |
| Career Guidance | No | Limited | Full |

## Duo Go

### Best For
- First-time mentorship seekers
- Testing the Space system
- Budget-conscious learners

### What You Get
| Feature | Included |
|---------|----------|
| Space access | Yes |
| Regular Nunu matching | Yes |
| Discord community | Yes |
| My Space dashboard | Yes |

### Price
Most affordable tier

### The Experience
- Match with community mentors
- Access to Matching Board
- Basic mentorship features
- Good starting point

## Decision Framework

### Choose Go if...
- "I just want to try it"
- "I'm on a budget"
- "I want to test the waters"
- "Community mentors are fine"

### Choose Run if...
- "I want quality mentorship"
- "I need professional guidance"
- "I want verified experts"
- "I'm serious about growth"

### Choose Fly if...
- "I want the absolute best"
- "I need strategic direction"
- "I want founder access"
- "Investment isn't a concern"

## Upgrading

### Can I Start with Go and Upgrade?
Yes! You can upgrade anytime:
- Go → Run: Pay difference
- Go → Fly: Pay difference
- Run → Fly: Pay difference

### How to Upgrade
1. Go to Shop → Duo Tickets
2. Select higher tier
3. System calculates difference
4. Pay and unlock immediately

---

## FAQ

### Can I downgrade?
Yes, at the end of your billing period.

### What if I cancel?
Access continues until billing period ends.

### Do I need Solo Traveler first?
Yes, Duo Tickets are add-ons to Solo Traveler.

### Can I switch between tickets?
Yes, upgrade or downgrade as needed.
