# Forum: House Rules

> **Section ID:** `forum-guidelines`
> **Parent:** Forum
> **Purpose:** Community guidelines and moderation

---

## TL;DR

Be nice. Write clear titles. Search before asking. Credit others. Don't spam.

---

## The Golden Rule

> **Would you want to receive this post?**
>
> If yes, send it. If no, revise it.

---

## Do This

### Be Nice
- Treat everyone with respect
- Assume good intentions
- Help newcomers feel welcome
- Celebrate others' wins

### Write Clear Titles
- Describe the topic clearly
- Avoid clickbait
- Include key context
- Make it searchable

### Search Before Asking
- Check if it's been answered
- Avoid duplicate posts
- Reference existing threads

### Tag Properly
- Use the right category
- Add relevant tags
- Help people find your post

### Credit Others
- Link to original sources
- Mention creators
- Don't claim others' work

---

## Don't Do This

### Spam
- No excessive self-promotion
- No repetitive posting
- No automated content
- No unsolicited ads

### Be Rude
- No personal attacks
- No harassment
- No discrimination
- No trolling

### Share Private Info
- No personal contact info
- No confidential data
- No doxxing
- Protect others' privacy

### Create Duplicates
- Search before posting
- Use existing threads
- Combine related topics

### Use Clickbait
- No misleading titles
- No sensationalism
- No false urgency
- Keep it honest

---

## Content Standards

### Acceptable
- AI/ML discussions
- Prompt engineering
- Tool recommendations
- Career questions
- Project showcases
- Learning resources

### Not Acceptable
- Off-topic content
- NSFW material
- Illegal activities
- Harmful content
- Misinformation

---

## Moderation

### What Happens If You Break Rules

1. **First offense:** Warning
2. **Second offense:** Temporary mute
3. **Third offense:** Longer suspension
4. **Severe/repeated:** Permanent ban

### Reporting

To report a post or comment:
1. Click the "..." menu on the content
2. Select "Report"
3. Choose a reason
4. Submit

All reports are reviewed by moderators.

---

## Earning Reputation

### How to Build Rep
- Post helpful content
- Answer questions thoroughly
- Get upvotes
- Be consistently helpful

### Benefits of High Rep
- Trusted member status
- Leaderboard visibility
- Community recognition

---

## Quality Checklist

Before posting, ask yourself:

- [ ] Is my title clear and descriptive?
- [ ] Did I search for existing answers?
- [ ] Is my content appropriate?
- [ ] Did I use the right category?
- [ ] Am I being respectful?
- [ ] Does this add value?
