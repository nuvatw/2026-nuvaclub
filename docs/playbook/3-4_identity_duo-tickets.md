# Identity: Duo Tickets

> **Section ID:** `identity-duo`
> **Parent:** Identity System
> **Prerequisite:** Solo Traveler subscription

---

## TL;DR

Duo Tickets unlock **Space** - the mentorship matching system. Three tiers: Go (basic), Run (certified), Fly (VIP/founder).

---

## What Are Duo Tickets?

Duo Tickets are add-ons to your Solo Traveler subscription that unlock:

1. **Space Access** - The mentorship matching platform
2. **Mentor Connections** - Find and work with Nunus (mentors)
3. **Discord Community** - Connect with other members
4. **My Space Dashboard** - Manage your mentorship relationships

---

## The Three Tiers

### Duo Go

**Best for:** First-time mentorship seekers

| Feature | Included |
|---------|----------|
| Space access | Yes |
| Regular Nunu matching | Yes |
| Discord community | Yes |
| My Space dashboard | Yes |
| Certified Nunu access | No |
| Founder access | No |

**Price:** Most affordable tier

---

### Duo Run

**Best for:** Serious learners wanting professional guidance

| Feature | Included |
|---------|----------|
| Everything in Go | Yes |
| Certified Nunu mentors | Yes |
| Skip the matching queue | Yes |
| Pro-level guidance | Yes |
| Founder access | No |

**Price:** Mid-tier

---

### Duo Fly

**Best for:** VIPs who want the best possible experience

| Feature | Included |
|---------|----------|
| Everything in Run | Yes |
| 1:1 with Founder (Shang-Zhe) | Yes |
| AI strategy consulting | Yes |
| Career guidance | Yes |
| Exclusive community | Yes |

**Price:** Premium tier

## How to Choose

### Pick Go if...
- You're new to mentorship
- You want to try the system
- Budget is a concern
- You're okay with community mentors

### Pick Run if...
- You want verified expertise
- You're serious about learning
- You want faster matching
- You need professional guidance

### Pick Fly if...
- You want the absolute best
- You need strategic career advice
- You want founder-level insights
- Investment isn't a concern

---

## The Simple Decision Tree

```
"I just want to try it" → Go
"I want quality mentorship" → Run
"I want the absolute best" → Fly
```

---

## How to Purchase

1. Make sure you have a Solo Traveler subscription
2. Go to **Shop** → **Duo Tickets**
3. Choose your tier (Go, Run, or Fly)
4. Complete payment
5. Access Space immediately

---

## Can I Upgrade?

Yes! You can upgrade from:
- Go → Run
- Go → Fly
- Run → Fly

You'll be charged the price difference.
